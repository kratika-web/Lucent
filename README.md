# Lucent – AI Mental Health Companion

Lucent is an AI-powered mental health companion that provides a safe space for students to talk, reflect, and understand their emotional patterns.

## Features

• Emotion-aware intake questions  
• Personalised mental health radar chart  
• Adaptive conversational support  
• Weekly AI-generated insights  
• Session memory and emotional trend tracking  

## Tech Stack

Python  
Transformers (Emotion detection model)  
Groq LLM (Llama 3.3)  
Gradio (User Interface)  
Matplotlib (Radar charts)

## How It Works

1. User enters their name.
2. The system asks 5 gentle intake questions.
3. Emotional distress is analyzed using a transformer model.
4. A radar chart profile is generated.
5. The AI companion adapts its conversation style based on distress levels.
6. Weekly insights summarize emotional patterns.
