# ✨ Lucent — Your Warm Mental Health Companion

A beautiful, empathetic AI companion for students.  
Built with Groq (Llama-3.3-70B), emotion detection, personalised radar profiles, conversation memory, and weekly insights.

### Features
- 5-question gentle intake assessment  
- Real-time mental health radar chart  
- Warm ongoing chat with memory  
- Weekly personalised insight  
- Calm cloudy-day UI
